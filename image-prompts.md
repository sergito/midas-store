## NOTAS PARA LA GENERACIÓN:

1. **Consistencia de Marca**: Todas las imágenes deben sentirse parte de la misma familia visual
2. **Colores**: Usar la paleta de colores del sitio cuando sea posible (#7FB448 como acento)
3. **Estilo**: Fotografía profesional de producto, limpia, moderna, premium
4. **Autenticidad**: Las fotos de productos "usados" deben verse honestas pero profesionales
5. **Calidad**: Siempre pedir resolución 8K, fotorrealista, ultra-sharp para poder redimensionar

## HERRAMIENTAS RECOMENDADAS:
- **Midjourney**: Excelente para product photography y hero images
- **DALL-E 3**: Bueno para composiciones específicas y control de texto
- **Stable Diffusion**: Buena opción gratuita con control fino
